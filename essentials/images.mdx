## About

Playpulse Panel is an open-source game server management panel designed to provide a high-performance, feature-rich, and user-friendly experience for managing game servers. It is developed by hexlorddev and built with a focus on security, efficiency, and extensibility.

## Technology Stack

Playpulse Panel leverages a modern and robust technology stack to deliver its powerful features:

| Technology         | Version    | Purpose          |
| :----------------- | :--------- | :--------------- |
| 🐘 **PHP**         | `8.1+`     | Core Runtime     |
| 🚀 **Laravel**     | `10.x`     | Framework        |
| 🗄️ **MySQL/PostgreSQL** | `8.0+/13+` | Database         |
| ⚡ **Redis**        | `6.0+`     | Caching          |
| 🔑 **JWT**         | Latest     | Authentication   |
